# ChefLab

Recipe management system — recipes, ingredients, and approval workflow.

## Project structure

- **backend/** — Django REST API (Python, Django, DRF, PostgreSQL, JWT)
- **frontend/** — Next.js UI (TailwindCSS, Axios, Tiptap)

## Run

1. **Backend:** `cd backend && source .venv/bin/activate && python manage.py runserver`
2. **Frontend:** `cd frontend && npm install && npm run dev` (see [frontend/README.md](frontend/README.md))

See [backend/README.md](backend/README.md) for backend setup.
